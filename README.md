# 👟 Adidas Sales Report — Power BI

An interactive sales analytics dashboard built with Microsoft Power BI to analyze 
Adidas product sales performance across regions, retailers, and product categories.

## 📊 Dashboard Overview

The dashboard visualizes key Adidas sales metrics through:

- **Sales by Retailer** — Clustered column chart comparing retailer-wise performance
- **Sales by Region** — Pie chart showing regional sales distribution
- **Profit Waterfall** — Waterfall chart tracking profit gains and losses over time
- **Units Sold by Product** — Clustered bar chart breaking down units per product category

## 🗂️ Dataset

📂 **Source:** [Kaggle — Adidas US Sales Dataset](https://www.kaggle.com/datasets/heemalichaudhari/adidas-sales-dataset)  
> Dataset includes sales data across retailers, regions, products, pricing, and profit metrics for Adidas US operations.

## 🛠️ Tools & Technologies

- Microsoft Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (Data Transformation)
